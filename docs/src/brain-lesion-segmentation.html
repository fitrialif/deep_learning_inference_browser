Brain Lesion Subregion Segmentation using Deep Learning

The model was trained to segment brain habitats (necrotic core, enhancing tumor, edema) on multi parametric MR images (T1 post contrast, T2 and Flair) using public data made available for the BRATS challenge. The trained model was imported in java script using keras.js
